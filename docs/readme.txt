Επιλογές Υλοποιήσης:

Η εργασία υλοποιήθηκε σε γλώσσα c++.
Χρησιμοποιήθηκε git(bitbucket) κατά τον σχεδιασμό της εργασίας.
Οι δομές που αναπτύχθηκαν για τις ανάγκες του 1ου μέρους είναι ένα hashtable με templates
για χρήση σε 2 διαφορετικές περιπτώσεις(queries,node index) και μια ουρά(bi-bfs).

Γράφος:Η αναπαράσταση του γράφου έγινε μέσω μιας κλάσης graph η οποία περιέχει 2 ζευγάρια
buffer-node_index και καλέι τις μεθόδους add,qeury για εισαγωγές κόμβων και επερωτήσεις.

buffer:Ο buffer είναι ένας μεγάλος πίνακας από list nodes ο οποίος έχει ένα default
αρχικό μέγεθος και αυξάνεται ανάλογα με τους κόμβους που προστίθενται.

node_index:Το node index έχει υλοποιηθεί με hashtable μιας και μετά από δοκιμές αποδείχθηκε
ότι η χρήση hashtable έδινε καλύτερους χρόνους.

queries:Για τα queries υλοποιήθηκε bi-directional bfs αλγόριθμος.Συγκεκριμένα, μετά από
δοκιμές αντικαταστάθηκε η αρχική υλοποίηση του πίνακα κλειστού συνόλου κόμβων που έχουν 
επισκεφτεί με ένα hashtable λόγων μικρότερης μνήμης και καλύτερων χρόνων που προσέφερε.
Επίσης, ο αλγόριθμος τροποποιήθηκε ώστε να τρέχει κάθε φορά από την μεριά που έχει
τους λιγότερους κόμβους μέσα στην ουρά για να επεκτείνει,επιτυγχάνοντας ταχύτερους χρόνους.

Η εργασία επιτυγχάνει χρόνο ~30s για το medium dataset στους υπολογιστές που δοκιμάστηκε.
